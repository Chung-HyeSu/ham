# 🐹 햄찌

키보드를 치면 같이 춤추는 Windows 데스크탑 햄스터 펫.

## 동작

- **타이핑 중** → 타이핑 속도에 비례해서 춤춤 (빠를수록 빠르게)
- **멈추면** → 정지
- **1분 이상 자리 비우면** → 알아서 자동 댄스
- **우클릭** → 종료
- **드래그** → 원하는 위치로 이동

## 실행 방법

### 1. Python 설치

https://www.python.org 에서 Python 3.8 이상 설치 (PATH 추가 체크 필수)

### 2. 첫 실행

`햄찌.vbs` 더블클릭

> 패키지 자동 설치 → 바탕화면 바로가기 생성 → 햄찌 실행

이후엔 바탕화면 **햄찌** 아이콘으로 실행.

### 3. 첫 실행 시 처리 시간

`ham.mp4`에서 배경을 제거하고 프레임을 추출합니다.  
처음 한 번만 시간이 걸리고, 이후엔 캐시(`cache/` 폴더)를 사용해 바로 실행됩니다.

## 필요 패키지

```
pip install pillow rembg opencv-python pynput
```

(`햄찌.vbs` 실행 시 자동 설치됨)

## 파일 구성

```
ham/
├── hamster.py        # 메인 코드
├── ham.mp4           # 햄찌 원본 영상
├── hamster_icon.ico  # 아이콘
└── 햄찌.vbs          # 실행 런처 (더블클릭)
```
